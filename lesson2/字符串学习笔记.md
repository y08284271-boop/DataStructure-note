# C++ String字符串学习笔记

## 1. 字符串声明和初始化

### 基本用法
```cpp
string s1;                          // 空字符串
string s2="hello world";            // 用字符串初始化
string s3("hello world");           // 用构造函数初始化
```

### 查看内容
```cpp
cout << s2 << endl;
cout << s3 << endl;
```

---

## 2. 字符串输入

### getline() 函数

#### 基本用法
```cpp
string s1;
getline(cin, s1);  // 输入整行
cout << s1 << endl;
```

#### 指定输入结束标志
```cpp
string s;
getline(cin, s, 'a');  // 以'a'作为输入结束标志
cout << s << endl;
```

## 3. 字符串大小

### size() 方法

```cpp
string s = "hello world";
```

## 5. 迭代器

### 迭代器基本操作
```cpp
string s = "abcd";

string::iterator it1 = s.begin();  // 开始迭代器
string::iterator it2 = s.end();    // 结束迭代器

cout << *it1 << endl;   // 解引用，输出第一个字符
it1++;                   // 迭代器后移
cout << *it1 << endl;   // 输出第二个字符
it1--;                   // 迭代器前移
cout << *it1 << endl;   // 回到第一个字符

cout << it2 - it1 << endl;  // 计算迭代器之间的距离
```

### 迭代器遍历
```cpp
for(auto it = s.begin(); it < s.end(); it++){
    cout << *it << endl;
}
```

## 6. push_back() - 尾部追加字符

### 基本用法

## 7. 字符串拼接

### 使用 += 操作符
```cpp
string s = "hello ";
s += "world";               // 将"world"追加到s后面
cout << s << endl;          // 输出：hello world
```

### 使用 + 操作符
```cpp
string s = "hello ";
cout << s + "world" << endl;  // 输出：hello world
```

### 字符串作为开头（需要先是string类型）
```cpp
string s = "hello ";
s = "world" + s;  // 错误！不能用C风格字符串加string
```

## 8. pop_back() - 删除末尾字符

### 单个删除
```cpp
string s = "hello";
s.pop_back();           // 删除最后一个字符'o'
cout << s << endl;      // 输出：hell
cout << s.size() << endl;  // 输出：4

s.pop_back();           // 再删除一个
cout << s << endl;      // 输出：hel
cout << s.size() << endl;  // 输出：3
```

### 循环删除至空字符串
```cpp
string s = "hello";
while(s.size() > 0){
    s.pop_back();
}
cout << s.size() << endl;  // 输出：0
```

## 9. insert() - 插入字符或字符串

### 基本用法
```cpp
string s = "hello";
string str = "xxx";

// 方式1：在位置3插入字符串对象
s.insert(3, str);

// 方式2：在位置3插入C风格字符串
s.insert(3, "xxx");

// 方式3：在位置3插入3个字符'x'
s.insert(3, 3, 'x');  // 输出：hexxxllo
cout << s << endl;
```

## 10. find() - 查找字符或子字符串

### 基本查找
```cpp
string s = "hello world hello everyone";
string str = "llo";

size_t n = s.find(str);  // 返回第一次出现的位置
cout << n << endl;  // 输出：2

n = s.find(str, n+1);  // 从n+1位置开始继续查找
cout << n << endl;  // 输出：9
```

### 指定查找长度
```cpp
string s = "hello world hello everyone";
string str = "world";

// find()的第三个参数是要查找的字符串长度
size_t n = s.find("world", 0, 3);  // 查找"world"的前3个字符
cout << n << endl;

n = s.find("everyone", n+1, 5);  // 查找"everyone"的前5个字符
cout << n << endl;
```

## 11. substr() - 字符串截取

### 基本用法
```cpp
string s = "hello world hello everyone";

// 无参数：返回整个字符串
string s1 = s.substr();
cout << s1 << endl;  // 输出：hello world hello everyone

// 一个参数：从指定位置开始到末尾
string s2 = s.substr(7);
cout << s2 << endl;  // 输出：world hello everyone

// 两个参数：从指定位置开始取指定长度
string s3 = s.substr(7, 6);
cout << s3 << endl;  // 输出：world（6个字符）
```

### 与find()结合使用
```cpp
string s = "hello world hello everyone";
size_t n = s.find("world");  // 查找"world"的位置
string s1 = s.substr(n, 10);  // 从该位置开始取10个字符
cout << s1 << endl;  // 输出：world hello
```

## 12. 字符串比较

### 比较大小
```cpp
string s1 = "abcd";
string s2 = "abbcdef";

if(s1 > s2){
    cout << ">" << endl;
} else {
    cout << "<=" << endl;
}
```

### 重要提示
⚠️ **两个C风格的字符串不能直接这样比较！**

只有string类型才能使用 `>`, `<`, `==` 等比较操作符。C风格字符串需要使用`strcmp()`函数进行比较。
